# SP500-Market-Analysis-Dashboard
Análisis S&P 500: Rendimiento y Riesgo
Objetivo
Analizar el comportamiento de las 10 empresas con mayor peso en el S&P 500 desde 2021 a 2026, enfocándose en el retorno acumulado y la volatilidad mensual.

Herramientas Utilizadas
Python (Pandas/Plotly): Limpieza de datos y feature engineering (Retorno Diario y Volatilidad).

Tableau: Visualización interactiva y análisis de tendencias.

Visualización del Dashboard
Conclusiones Clave
Identificación de picos de volatilidad en el sector tecnológico.

Comparativa de volumen vs. movimiento de precios.
